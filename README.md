 Lezzetli Yemek Tarifleri 

Bu proje, HTML ve CSS kullanılarak geliştirilmiş basit bir yemek tarifleri web sitesidir.  


 Proje İçeriği

Web sitesi aşağıdaki sayfalardan oluşmaktadır:

- Ana Sayfa
- Tarifler
- Tablolar
- İletişim
- Hakkımızda

Her sayfa birbirine navigasyon menüsü ile bağlıdır.


  Projenin Amacı

- HTML etiketlerini doğru ve anlamlı kullanmak
- CSS ile sayfa tasarımı ve düzen oluşturmak
- Çok sayfalı web sitesi yapısını öğrenmek
- Header, nav, footer gibi temel bölümleri uygulamak

 Projeyi Çalıştırma

1. Projeyi bilgisayarınıza indirin veya klonlayın
2. `index.html` dosyasını tarayıcıda açın
3. Site kullanıma hazırdır
